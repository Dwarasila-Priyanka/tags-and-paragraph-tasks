What is Internet?

The Internet is a worldwide system of connected computer networks that allows people to communicate, share information, and access services like websites, email, and social media.

What is browser?

A browser is a software application that allows you to access and view websites on the Internet.

EX:Google Chrome,Microsoft Edge,Opera

What is domain?

A domain is the human-readable address of a website on the Internet

Domains have different parts: 1.Top-Level Domain (TLD): like .com, .org, .net 2.Second-Level Domain: like google in google.com 3.Subdomain (optional): like www

For example, in the web address www.google.com com = Top-Level Domain example = Second-Level Domain www = Subdomain

What is IP address?

An IP address (short for Internet Protocol address) is a unique number assigned to each device connected to the Internet. It works like a home address, but for your computer or phone, allowing data to be sent to the correct place.

What is URL?

A URL (Uniform Resource Locator) is the web address you type into a browser to visit a specific page or resource on the Internet.

EX:https://www.example.com/page1

What is bug?

A bug is an error, flaw, or fault in a software program or system that causes it to behave unexpectedly or incorrectly.

What is debug?

Debug is the process of finding and fixing bugs or errors in a software program.

what is https?

HTTPS stands for HyperText Transfer Protocol Secure. It is the secure version of HTTP, the protocol used to transfer data between your browser and a website.

What is http?

HTTP stands for HyperText Transfer Protocol.It is the foundation of data communication on the web, used by browsers to request and receive web pages from servers.

What is Mark up language?

A markup language is a type of computer language used to structure, format, and present content—especially text—on web pages and documents. EX:HTML,XML

What is Programming language?

A programming language is a formal language used to write instructions that a computer can understand and execute to perform specific tasks or solve problems. EX:Python,java

What is XHTML?

XHTML stands for Extensible HyperText Markup Language. It is a web coding standard that is a stricter version of HTML , combining the flexibility of XML (Extensible Markup Language) with the simplicity of HTML.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create and structure content on the web.
